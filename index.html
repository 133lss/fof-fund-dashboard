TWR 时间加权收益率 图表1主线
定义：Time-Weighted Return，在每个资金进出时点切断，分段计算子区间收益率后几何连乘。
核心价值：完全消除分批建仓对收益率曲线的扭曲。本FOF分4笔建仓（100万→300万→400万→500万），简单收益率在每次新增投入时会出现阶梯式跳变，TWR消除这一影响，如实反映投资管理能力。
计算方法：在每个日期点，若累计投入发生变化（有新资金进入），则用 (当日市值 - 新增现金流) / 前日市值 - 1 计算子区间收益率；若无资金变动，则直接用 当日市值 / 前日市值 - 1。所有子区间收益率连乘得到累计TWR。
子区间收益率 rᵢ = (NVᵢ - CFᵢ) / NVᵢ₋₁ - 1　|　TWR = ∏(1 + rᵢ) - 1
解读：TWR ≥ 0 表示投资组合在扣除资金进出影响后获得正收益。TWR越高，说明基金经理的择时选股能力越强。与简单收益率对比可看出资金进出节奏对整体收益的影响程度。
注：本系统TWR基于费后净值计算，已扣除业绩报酬预扣。图表1中绿色实线为TWR，虚线为简单收益率（未调整资金时点），淡虚线为费前收益率。
XIRR 年化内部收益率 关键指标
定义：Extended Internal Rate of Return，考虑每笔现金流的时间价值，通过Newton-Raphson迭代法求解使所有现金流净现值(NPV)为零的折现率，并年化处理。
核心价值：同时考虑了「投入金额大小」和「资金占用时间长短」，是衡量资金使用效率的金标准。对比TWR（只看管理能力不看资金节奏），XIRR反映的是投资者实际拿到的年化回报。
计算方法：每笔申购为负现金流（资金流出），期末费后总市值为正现金流（资金回流），求解折现率r使NPV=0。
NPV = Σ CFᵢ / (1 + r)^((dateᵢ - date₀) / 365.25) = 0　→　用Newton法迭代求r
解读：XIRR > 0 表示投资整体盈利。XIRR可直接与银行理财、基金产品年化收益率横向比较。本FOF的XIRR反映的是考虑了4笔分批建仓时间差异后的真实年化回报。
注：XIRR使用每笔子基金的实际申购日期和金额构建现金流，期末市值为最新一期费后总市值。迭代精度1e-7，最大100次迭代。
MOIC 投资倍数 关键指标
定义：Multiple on Invested Capital，当前费后总市值 ÷ 累计投入金额，衡量投入资本的增值倍数。
核心价值：最直观的投资回报度量——投入1块钱变成了几块钱。不考虑时间因素，纯粹看资本增值结果。
计算方法：
MOIC = 费后总市值 / 累计投入
解读：MOIC = 1.0 表示盈亏平衡；MOIC = 1.5 表示投入翻1.5倍（赚50%）；MOIC = 2.0 表示翻倍。与XIRR互补使用：MOIC看「赚了多少倍」，XIRR看「以什么效率赚到的」。例如同样1.5倍MOIC，1年达到和3年达到的XIRR完全不同。
注：MOIC基于费后净值计算，已扣除高水位法预提的业绩报酬。
简单收益率 参考指标
定义：当前费后总市值 ÷ 累计投入 - 1，未对资金进出时点做任何调整。
局限：分批建仓时，新增资金会立即改变分母（累计投入），导致收益率曲线出现阶梯式跳变。例如本FOF在第41个日期点（2025-08-27）新增200万投入时，收益率从约32%骤降到约20%，这不是亏损，而是分母增大导致的数学效应。
保留原因：作为未经调整的原始数据参考，与TWR对比可直观看出资金进出对收益率的影响幅度。卡片中标注"未调整资金时点"。

指标对比与选择指南 总结
为什么需要多种指标？没有单一指标能完美衡量投资表现，不同指标反映不同维度：
TWR　→ 评估基金经理管理能力（剥离资金进出影响）
XIRR → 评估投资者实际年化回报（考虑资金时间价值）
MOIC → 评估资本增值绝对倍数（最直观的盈亏度量）
简单收益率 → 原始参考（未调整，展示资金进出影响）
本FOF建议阅读顺序：①先看MOIC了解整体盈亏倍数 → ②看XIRR了解年化效率 → ③看TWR曲线了解管理能力趋势 → ④对比简单收益率理解资金节奏影响。
